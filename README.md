# 🤖 Telegram Bot Hosting Panel

A powerful Telegram bot hosting panel that allows users to deploy and manage their own Telegram bots (PHP & Python).

## Features
- ✅ Deploy PHP & Python bots
- ✅ Manage bots (Start/Stop/Restart)
- ✅ User system with limits
- ✅ Admin panel
- ✅ File upload support
- ✅ ZIP file extraction

## Commands
- `/start` - Start the bot
- `🤖 NEW BOT` - Deploy a new bot
- `📋 MY BOTS` - View your bots
- `📊 STATS` - View statistics
- `👤 PROFILE` - View profile

## Deployment
1. Get bot token from @BotFather
2. Click "🤖 NEW BOT"
3. Send your token
4. Upload your bot file (.php or .py)
